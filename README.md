# Interactive JavaScript Practice Page

A simple, interactive web page built with **HTML**, **Tailwind CSS**, and **JavaScript** to practice basic DOM manipulation, event handling, and simple math operations.  

---

## 📝 Features

1. **User Info Display**  
   Displays your name and registration number in a styled info card.

2. **Interactive Header**  
   - Click the **Update Title** button to modify the main page title dynamically.

3. **Button Interaction**  
   - A simple button that provides feedback when pressed using JavaScript.

4. **Math Tool / Calculator**  
   - Perform basic arithmetic operations: addition (+), subtraction (−), multiplication (×), and division (÷).  
   - Enter two numbers and click an operation to see the result.

5. **Daily Task Manager**  
   - Add tasks to a to-do list dynamically.  
   - Tasks appear below the input field in a styled container.

---

## 🎨 Technologies Used

- **HTML5** – Structure of the page  
- **Tailwind CSS** – Modern utility-first CSS for styling  
- **JavaScript** – DOM manipulation, events, and calculator logic
